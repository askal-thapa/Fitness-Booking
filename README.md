# Gym Trainer Booking Prototype

This is a premium, British-inspired Gym Trainer Booking Application built with Next.js, Tailwind CSS, and TypeScript. 

Since this is a front-end prototype and the backend routing/authentication is simulated, you can explore the various views directly by navigating to the following URLs in your browser:

## 🧭 Application Directory

Make sure the development server is running (`npm run dev`), then click or paste these URLs into your browser:

### Public Pages
- **Landing Page:** [http://localhost:3000](http://localhost:3000)
- **Login & Registration:** [http://localhost:3000/login](http://localhost:3000/login)

### Fitness User Flows
- **Main Dashboard:** [http://localhost:3000/dashboard](http://localhost:3000/dashboard)
- **Browse Trainers:** [http://localhost:3000/dashboard/trainers](http://localhost:3000/dashboard/trainers)
- **Trainer Profile Details:** [http://localhost:3000/dashboard/trainers/1](http://localhost:3000/dashboard/trainers/1) *(Simulated IDs: 1 to 6)*
- **Book a Session:** [http://localhost:3000/dashboard/trainers/1/book](http://localhost:3000/dashboard/trainers/1/book)
- **Manage Bookings:** [http://localhost:3000/dashboard/bookings](http://localhost:3000/dashboard/bookings)
- **User Profile Settings:** [http://localhost:3000/dashboard/profile](http://localhost:3000/dashboard/profile)

### Personal Trainer Flows
- **Trainer Command Center:** [http://localhost:3000/trainer/dashboard](http://localhost:3000/trainer/dashboard)

## ✨ Global Features
- **Floating AI Assistant:** A globally persistent chat widget accessible from the bottom right corner of every page in the application.

---

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
